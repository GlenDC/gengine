gengine
=======

An experimental game engine

## Status
Branch 'dev' : [![Build Status](https://travis-ci.org/gogoprog/gengine.svg?branch=dev)](https://travis-ci.org/gogoprog/gengine)

## Features
 * Emscripten/WebGL support
 * Lua scripting
 * Gameplay components
   * Sprite
   * Camera
   * Mouseable
   * Animated sprite
